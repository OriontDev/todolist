# 📝 To-Do List App

A to-do list app built with **React** and **Vite**.

## 🚀 Features

- Dark and Light Mode toggle 🌙☀️  
- Add and delete categories  
- Animated pop-in  
- Move categories left and right  
- Delete mode with warning border  
- Responsive design

## 🧠 Tech Stack

- React  
- Vite  
- JavaScript  
- CSS
